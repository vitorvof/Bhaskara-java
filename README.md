# 🧮 Calculadora de Bhaskara

Este projeto tem como objetivo calcular as raízes de uma equação do segundo grau utilizando a fórmula de Bhaskara.

## 📌 Sobre

A equação do segundo grau possui o formato:

ax² + bx + c = 0

Através da fórmula de Bhaskara, é possível encontrar as raízes dessa equação:

Δ = b² - 4ac  
x = (-b ± √Δ) / (2a)

---

## 🚀 Funcionalidades

- Entrada dos valores de **a**, **b** e **c**
- Cálculo do **delta (Δ)**
- Verificação do tipo de raízes:
  - Duas raízes reais
  - Uma raiz real
  - Nenhuma raiz real
- Exibição do resultado
